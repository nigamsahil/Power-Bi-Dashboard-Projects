# HR Analytics Dashboard

This Power BI dashboard analyzes key HR metrics such as employee demographics, attrition, salary structure, department comparison, and performance indicators.

## Features:
- Employee Headcount Overview
- Attrition & Retention Analysis
- Department-wise Employee Distribution
- Job Role & Salary Insights
- Performance Evaluation Patterns

## Tools Used:
- Power BI Desktop
- Excel / CSV Dataset
- DAX Functions
